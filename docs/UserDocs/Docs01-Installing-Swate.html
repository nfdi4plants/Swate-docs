Installing Swate

last updated at 2022-09-19

We currently offer different routes of installing Swate.

Browser Installation

If you want to jump right into Swate and start testing the environment and features without much fuzz, we recommend using Swate in your Browser. This method uses the web version of Microsoft Excel which can be used for free with a free Microsoft account.

Desktop Installation
